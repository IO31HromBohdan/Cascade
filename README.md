# Cascade — планувальник завдань

## Мета проєкту

Cascade — це веб-застосунок для планування щоденних та довгострокових завдань. Користувач може фіксувати задачі, групувати їх за днями, тижнями та місяцями, додавати нотатки та відстежувати прогрес виконання.

## Команда проєкту

Проєкт виконується індивідуально.  
Роль розробника: проєктування, розробка, тестування та документування застосунку.

## Цільова аудиторія

- Студенти, які ведуть список лабораторних робіт, дедлайнів та консультацій.
- Фахівці, яким потрібно планувати робочі задачі та зустрічі.
- Користувачі, які хочуть мати простий “щоденник завдань” у веб-інтерфейсі.

## Основний функціонал (MVP)

- Створення, редагування та видалення задач.
- Прив’язка задач до конкретної дати.
- Режими перегляду:
  - **Today** — задачі на поточний день.
  - **Week** — задачі на поточний тиждень.
  - **Month** — загальний огляд задач по днях місяця.
- Відмітка задач як виконаних.
- Текстові нотатки до дня або окремої задачі.

## Попередній технологічний стек

- **Frontend**: React / Next.js, TypeScript.
- **Backend**: Node.js (наприклад, NestJS або Express).
- **База даних**: PostgreSQL.
- **Інші інструменти**: Docker, GitHub Actions для CI, ESLint, Prettier.

## Подальший розвиток

- Теги задач (навчання, робота, особисте тощо).
- Нагадування та нотифікації.
- Авторизація користувачів і збереження даних у хмарі.
- Фільтрація та пошук задач за тегами, статусом та часовими інтервалами.
